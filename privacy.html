eSheep+用户支持

隐私政策

更新日期：2026 年 7 月 11 日

eSheep+ 不投放广告，不出售个人信息，也不进行跨 App 或跨网站跟踪。

1. 我们处理的数据

  • 账号信息：邮箱地址、Supabase 用户 ID、登录会话。
  • 羊场数据：牧场、成员角色、羊只档案、圈舍、投喂、繁殖、称重、健康、批次和历史事件。
  • 用户内容：用户主动添加的羊只照片、导入文件、AI 问题和所选图片。
  • 本机设置：DeepSeek 和 MiMo API Key、聊天记录及本地默认牧场数据。

2. 数据存储与用途

邮箱和用户 ID 用于注册、登录和权限验证。云端牧场数据（包括写入 FarmData 的羊只照片）存储在 Supabase，用于多设备同步、成员协作、备份恢复和 App 核心功能。登录令牌与用户填写的 AI API Key 保存在 Apple 钥匙串中。

3. AI 服务

当用户主动使用 DeepSeek 时，问题文本、必要的对话上下文，以及工具查询得到的必要羊场数据会发送到 DeepSeek 以生成回答。当用户主动选择图片分析时，问题和所选图片会发送到 MiMo；MiMo 的观察结果和原始问题可能继续发送到 DeepSeek。eSheep+ 不会把用户的 API Key 上传到我们的服务器。

4. 语音与图片

语音输入可能使用 Apple Speech，也可以使用下载到本机的 Whisper 模型。Whisper 录音在本机临时处理并在转写后删除。Apple Speech 的处理受 Apple 相应隐私规则约束。只有用户主动选择或拍摄的图片才会用于羊只档案或 AI 图片分析。

5. 第三方服务

  • Supabase：账号认证、数据库与云端同步。
  • DeepSeek：用户主动发起的文本 AI 请求与工具结果分析。
  • MiMo：用户主动发起的图片和多模态分析。
  • Apple：钥匙串、语音识别、系统权限和 App 分发。

6. 保存期限与账号删除

云端数据在账号或牧场有效期间保存。用户可在 App 的“个人中心 → 账户与隐私”申请删除账号。场主需要先转让或明确删除名下牧场。申请提交后，账号进入 7 天撤销期并暂停访问牧场云端数据；期间重新登录可撤销。到期后账号及其个人成员关系永久删除。用户已转让的牧场继续归新场主和其他成员使用。

删除账号不会自动删除设备上的本地默认牧场或用户自行保存的 AI API Key；用户可在 App 中单独清理这些内容。

7. 用户选择与安全

用户可以拒绝麦克风、语音识别、相机或照片权限，相应功能将不可用但不会影响其他核心功能。我们使用角色权限、行级安全策略、HTTPS 和系统钥匙串保护数据。
