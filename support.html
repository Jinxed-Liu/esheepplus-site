账号与牧场

  • 登录后必须新建牧场，或输入场主提供的邀请码加入牧场，才会加载云端数据。
  • 场主负责成员邀请、角色调整、牧场改名和受保护记录删除。
  • 默认牧场是设备本地空间，不参与云端同步。

删除账号

打开“个人中心 → 账户与隐私 → 申请删除账号”。如果仍是场主，请先把每个牧场转让给现有成员，或先导出 JSON 备份后明确删除牧场。删除申请有 7 天撤销期；期间重新登录即可看到撤销入口。

备份与恢复

在“更多 → 本地备份”导出 JSON 文件。恢复前确认当前所选牧场，避免把备份恢复到错误的本地空间。云端数据发生异常时不要用陈旧本地缓存覆盖，应先联系支持确认。

AI 助手

DeepSeek 和 MiMo 使用用户自己的 API Key。Key 只保存在本机。文本 AI、图片 AI 和本地查询的可用性取决于相应 Key、网络和当前角色权限。
