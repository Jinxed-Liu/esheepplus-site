账号与牧场

从新建或加入开始

  • 登录后，新建牧场或输入场主提供的邀请码加入牧场，才会加载对应云端数据。
  • 场主负责成员邀请、角色调整、牧场改名和受保护记录删除。
  • 默认 main 牧场是设备本地空间，不参与云端同步。

同步与协作

先确认当前牧场和角色

在多设备使用时，先确认两台设备登录的是同一账号、选中的是同一个牧场，并且成员角色具有相应权限。云端异常时，请不要用陈旧的本地缓存覆盖云端数据；先联系支持确认。

如果出现“无权访问”或“无法加载”，请记录牧场名称、账号邮箱和错误发生时间后发送至支持邮箱。

备份与恢复

保留一份可验证的本地备份

在“更多 → 本地备份”导出 JSON 文件。恢复前请确认当前所选牧场，避免把备份恢复到错误的本地空间。导入或恢复后，请核对羊只数量、圈舍与最近记录。

AI 助手

使用你自己的 API Key

DeepSeek 和 MiMo 使用用户自己的 API Key，Key 仅保存在本机钥匙串。文本 AI、图片分析和本地查询能否使用，取决于 Key、网络和当前牧场角色权限。你主动选择分析的图片会发送给 MiMo；回答所需的问题和羊场数据会发送给 DeepSeek。

删除账号

七天撤销期内可以恢复

在“个人中心 → 账户与隐私 → 申请删除账号”发起。若仍是场主，请先逐个将牧场转让给现有成员，或导出 JSON 备份后输入牧场名称确认删除该牧场。系统不会静默级联删除你拥有的牧场。

提交申请后会退出登录，云端牧场访问随即暂停。重新登录会显示删除倒计时和撤销入口；七天内撤销即可恢复。详见 隐私政策。
